# Contributing to Wallet Rebalancer

First off, thanks for considering contributing!

## How to Contribute
1. **Report bugs** by opening a GitHub issue. Please include reproducible steps.
2. **Request features** by opening an issue and explaining the use case.
3. **Submit Pull Requests** to fix issues or implement improvements.

## Development Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run tests to ensure everything is working:
   ```bash
   npx hardhat test
   ```

## Commit Guidelines
- Use conventional commits if possible.
- Ensure your tests pass before pushing.
- Keep PRs focused on a single concern.
